CSS란?

CSS란 Cascading style Sheets의 약자로, HTML같은 문서의 스타일을 꾸밀 때 사용하는 스타일 시트 언어입니다.
HTML이 문서의 뼈대라면 CSS는 문서의 꾸밈을 담당합니다.

CSS 사용법

  • CSS는 HTML 태그 안에 style을 넣어서 사용할 수 도 있습니다.
    예제 -> <span style="color:red;">빨간색</span> = 빨간색

  •  , 를 사용하면 중복되는 태그들을 사용하는 속성들을 묶을 수 있다.
    예제 -> <style> h1,a { } </style>
