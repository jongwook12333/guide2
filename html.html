HTML이란?

HTML이란? Hyper Text Markup Language 약어로 HyperText(웹 페이지에서 다른 페이지로 이동할 수 있도록 하는 것) 기능을 가진 문서를 만드는 언어입니다.
다시 말해, 구조를 설계할 때 사용되는 언어로 hyper link 시스템을 가지고 있으며, 흔히 말하는 웹 페이지를 위한 마크업 언어이며 문서의 뼈대라고 할 수 있습니다.

자주 사용하는 태그들

  • <a> = 링크태그이며 href,title등 과 같은 속성들과 같이 사용
  • &nbsp; = 공백
  • <br> = 줄 바꿈
  • <p> = 단락
  • <u> = 밑줄
  • <center> = 가운데 정렬
  • <img src="_._"> = 이미지
  • <ol> = li의 부모태그(숫자표시)
  • <ul> = li의 부모태그(숫자미표시)
  • <li> = 목차
  • <!-- --> = 주석
  • <div> = 의미가 없는 block(줄바꿈)이 기본인 태그
  • <span> = 의미가 없는 inline(줄바꿈 없앰)이 기본인 태그
  • <input type="button" value="이름값" [JavaScript]=""> = 버튼을 만드는 태그
  • <button> = css가 적용 가능한 버튼
  • <hr> = 선긋기
